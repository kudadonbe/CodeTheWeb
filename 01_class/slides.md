## 🧱 What is a Website Made Of?

### Websites use 3 languages:

- 🧱 **HTML** = Structure  
- 🎨 **CSS** = Style  
- ⚙️ **JavaScript** = Behavior  

> Think of a webpage like a house:
> - HTML is the frame 🏗️  
> - CSS is the paint & design 🎨  
> - JS brings it to life ⚡
